# Introduction

This project is a tool intended to support an RPG game. 

Whith this tool you can manage:

- Game systems
- Charactere sheets
- Campaign and adventure creation
- Adventure execution

## Techs

This project currently runs with:

- Angular v~7.2.0
- Electron v4.0.1
- Electron Builder v~20.36.2
- SQLite v^4.0.6
- TypeOrm v^0.2.11
